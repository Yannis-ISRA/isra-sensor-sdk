# Quickstart (EN)

This guide helps you quickly start with the ISRA SDK.

## 1) Prerequisites

- Python 3.10+
- A compatible UHF reader
- An ISRA sensor (known EPC)

## 2) Installation

```bash
pip install pyserial
```

## 3) First test

```bash
python ../examples/read_to_json.py
```
